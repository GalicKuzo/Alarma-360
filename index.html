FUNCIONAMIENTO

Este sistema de alarma utiliza un sensor de movimiento (PIR) conectado a un ESP32 para detectar intrusos. Cuando se activa el sensor, se enciende una alarma sonora (buzzer) y el estado se muestra en esta página web. Si la alarma está activa, puedes desactivarla manualmente desde el botón de "APAGAR" en la página. El sistema también muestra el estado actual de la alarma en tiempo real, permitiendo un monitoreo sencillo y accesible a través de cualquier dispositivo en la misma red WiFi.
